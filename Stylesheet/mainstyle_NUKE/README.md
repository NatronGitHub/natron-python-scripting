# mainstyle_NUKE

Mimics Nuke look.

**Note:** This stylesheet is outdated and will probably break if used with Natron v2.5+ and/or Qt5+.

## Windows : 
open Natron
>Edit > Preferences > Appearance
>>Stylesheet file (.qss) : set path to 'mainstyle_NUKE.qss' file.
>>>Font : Segoe UI
>>>>Font size : 9

## Linux/OSX :
open Natron
>Edit > Preferences > Appearance
>>Stylesheet file (.qss) : set path to 'mainstyle_NUKE.qss' file.
>>>Font size : 9
